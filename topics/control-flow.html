🧠 Chapter 3: Control Flow Statements in Python

Python is like a smart robot 🦾 — but to make decisions, it needs your instructions.
That’s where Control Flow Statements come in — they let you tell Python what to do, when to do it, and how many times.

🎯 What are Control Flow Statements?

  • Make decisions (if, else, elif)
  • Repeat actions (while, for)
  • Jump over code (break, continue, pass)

🤖 1. Conditional Statements – Making Decisions

📌 Syntax:

if condition:
    # run this block
elif another_condition:
    # run this if above fails
else:
    # fallback code

🎮 Real-Life Example:

age = 18
if age >= 18:
    print("You can vote! 🗳️")
else:
    print("Sorry, you are too young 😢")

🧪 Another Example:

marks = 85
if marks >= 90:
    print("Grade: A+ 🏆")
elif marks >= 75:
    print("Grade: A 🎉")
elif marks >= 60:
    print("Grade: B 🙂")
else:
    print("Grade: Needs Improvement 😐")
😂 Funny Analogy:
If your mom is home, you eat good food.
Else if your dad is home, you order pizza.
Else, you starve and eat Maggi. 🍜

🔁 2. Loops – Doing Things Repeatedly

🌀 A. while Loop

Repeats a block as long as a condition is true.

count = 1
while count <= 5:
    print("Count:", count)
    count += 1

Output:
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

🧠 Real-Life Analogy:
Imagine eating Laddus until the box is empty.
As long as the box has Laddus, you keep eating! 😋

🔂 B. for Loop

Used to iterate over a sequence (like a list, string, range).

for i in range(1, 6):
    print("This is line", i)

Output:
This is line 1
This is line 2
This is line 3
This is line 4
This is line 5

📚 Looping Through a List:

fruits = ["apple", "banana", "mango"]
for fruit in fruits:
    print("I like", fruit)

🛑 3. Loop Control Statements

⛔ break – Exit the loop immediately

for i in range(1, 10):
    if i == 5:
        break
    print(i)

Output: 1 2 3 4

⏩ continue – Skip to next iteration

for i in range(1, 6):
    if i == 3:
        continue
    print(i)

Output: 1 2 4 5

😶 pass – Do nothing (a placeholder)

for i in range(3):
    pass  # do nothing for now

Used when you write code but want to fill it later.

🧪 Practice Example: Guess the Number 🎲

secret = 7
guess = int(input("Guess a number between 1 and 10: "))
if guess == secret:
    print("🎉 You guessed it right!")
elif guess < secret:
    print("Too low! Try again.")
else:
    print("Too high! Try again.")

2. What will range(3) return?

  • a) [0, 1, 2, 3]
  • b) [1, 2, 3]
  • c) [0, 1, 2]

3. Which keyword exits a loop early?

  • a) exit
  • b) break
  • c) stop

4. What does pass do in Python?

  • a) Skips the loop
  • b) Does nothing
  • c) Ends the loop
