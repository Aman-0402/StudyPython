🧰 What is SQLite?

  • Built-in relational database in Python
  • Stores data in .db files
  • Perfect for learning and small projects

🔌 Step-by-Step: Connecting Python with SQLite

  1. Import sqlite3 module
    import sqlite3
  2. Connect to a database (or create it)
    conn = sqlite3.connect('students.db')
  3. Create a cursor object
    cur = conn.cursor()
  4. Create a table 
    cur.execute('''
CREATE TABLE IF NOT EXISTS students (
    id INTEGER PRIMARY KEY,
    name TEXT,
    marks INTEGER
)
''')
  5. Insert data
    cur.execute("INSERT INTO students (name, marks) VALUES (?, ?)", ("Aman", 95))
    conn.commit()
  6. Read data
    cur.execute("SELECT * FROM students")
rows = cur.fetchall()
for row in rows:
    print(row)
  7. Close the connection
    conn.close()

📦 Full Example

import sqlite3
conn = sqlite3.connect('school.db')
cur = conn.cursor()
cur.execute('CREATE TABLE IF NOT EXISTS student (id INTEGER PRIMARY KEY, name TEXT, grade INTEGER)')
cur.execute('INSERT INTO student (name, grade) VALUES (?, ?)', ('Riya', 88))
conn.commit()
cur.execute('SELECT * FROM student')
print(cur.fetchall())
conn.close()
