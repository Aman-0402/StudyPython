🧠 Chapter 4: Functions & Modules in Python

Imagine writing a recipe. You write the steps once and reuse them every time you cook the dish.
✅ That’s exactly what functions do in Python!

🍳 1. What is a Function?

A function is a block of reusable code that performs a specific task.

📌 Why Use Functions?

  • Avoid writing the same code again & again
  • Make your code clean, organized, and modular
  • Easy to debug and test

🧾 Defining a Function

def greet():
    print("Hello, Python Learner! 👋")

greet()  # calling the function

Output: Hello, Python Learner! 👋

🧃 2. Functions with Parameters

Functions can accept inputs (called parameters) and act on them.

def greet(name):
    print("Hello,", name)

greet("Aman")

Output: Hello, Aman

🔙 3. Return Statement

A function can also return a value using the return keyword.

def add(a, b):
    return a + b

result = add(3, 5)
print("Sum is:", result)

😄 Funny Example:

def coffee_machine(name):
    print(name, "your coffee is ready ☕️")

coffee_machine("Aman")

⚙️ 4. Types of Functions

TypeDescription

⚡ 5. Lambda (Anonymous) Functions

Used for short and simple functions.

square = lambda x: x * x
print(square(5))

Output: 25

📦 6. What is a Module?

A module is a Python file (.py) that contains functions, classes, or variables — basically, reusable code.
You can import a module in your project and use its features.

🔌 Importing a Module

import math
print(math.sqrt(16))

Output: 4.0

🌟 Common Python Modules

ModulePurpose
mathMath functions like sqrt, pow
randomRandom numbers and choices
datetimeDate and time
osFile and directory operations
sysSystem-specific parameters

🎲 Example: Using random Module

import random
number = random.randint(1, 10)
print("Random number:", number)

🧪 Practice Task 1: Calculator Function

def calculator(a, b, operation):
    if operation == 'add':
        return a + b
    elif operation == 'sub':
        return a - b
    elif operation == 'mul':
        return a * b
    elif operation == 'div':
        return a / b
    else:
        return "Invalid Operation"

print(calculator(10, 5, 'add'))  # Output: 15
