✅ 1. Introduction to Functions

🔹 What is a Function?

A function is a reusable block of code that performs a specific task.

🔹 Why Use Functions?

  • To organize code.
  • To avoid repetition.
  • To break problems into smaller pieces.

🧠 Syntax of a Function in Python:

def function_name(parameters):
    # block of code
    return result

✅ Example 1:

def copy():
    print("Don't Copy Me Idiot Try ur own example")

copy()  # Calling the function

✅ Example 2: Function with Parameters

def add(a, b):
    return a + b

result = add(5, 3)
print("Sum is:", result)

✅ 2. Function Parameters and Arguments

🔹 What are Parameters?

Parameters are the variables listed inside the parentheses in the function definition.
Arguments are the values you pass to the function when you call it.

✅ 3. The return Statement in Python Functions

🔹 What is return?

The return statement is used to send a result back to the place where the function is called.
It ends the function execution immediately.

✅ Example 1: Simple Return

def square(num):
    return num * num

result = square(4)
print("Square is:", result)

🟩 Output: Square is: 16

✅ Example 2: Return Multiple Values

def get_values():
    return 10, 20, 30

a, b, c = get_values()
print(a, b, c)

🟩 Output: 10 20 30

✅ Example 3: Return vs Print

def show():
    print("Hello")

def give():
    return "Hello"

x = show()     # prints Hello, but x is None
y = give()     # does not print but y gets the value "Hello"

print("x:", x)  # Output: x: None
print("y:", y)  # Output: y: Hello

✅ 5. Lambda (Anonymous) Functions

🔹 What is a Lambda Function?

A lambda function is a small anonymous function — meaning it doesn’t have a name like regular functions created with def.

📌 Syntax:

lambda arguments: expression

✅ It can take any number of arguments, but only one expression.

✅ Example 1: Simple Lambda Function

square = lambda x: x * x
print(square(5))  # Output: 25

✅ Example 2: Lambda with Multiple Arguments

add = lambda a, b: a + b
print(add(3, 4))  # Output: 7

✅ Example 3: Using Lambda with map()

nums = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, nums))
print(squared)  # Output: [1, 4, 9, 16]

✅ Example 4: Using Lambda with filter()

nums = [5, 8, 12, 3, 7]
even = list(filter(lambda x: x % 2 == 0, nums))
print(even)  # Output: [8, 12]

✅ Lambda vs def

✅ 6. Modules in Python

🔹 What is a Module?

A module is a file containing Python code (functions, variables, classes) that you can import and reuse in other programs.

📁 Think of a module as a toolbox — you import it when you need its tools (code).

🔹 Why Use Modules?

  • Reusability: Write once, use anywhere.
  • Organization: Keep code clean and separate.
  • Standard Library: Use built-in powerful features like math,random,datetime.

✅ Types of Modules

TypeExamples
Built-in Modulesmath, random, os, datetime
User-definedYour own .py files
Third-partynumpy, pandas, flask, etc. (via pip)

✅ Example 1: Import a Built-in Module

import math

print(math.sqrt(16))  # Output: 4.0

✅ Example 2: Import Specific Function

print(pi)        # Output: 3.14159...
print(pow(2, 3)) # Output: 8.0

✅ Example 3: Create and Use a User-defined Module

✅ Built-in Modules in Python

These modules come pre-installed with Python. You just import them and start using — no installation needed.

📦 1. math Module

Used for mathematical operations.

import math

print(math.sqrt(25))         # Square root: 5.0
print(math.ceil(4.2))        # Ceiling value: 5
print(math.floor(4.9))       # Floor value: 4
print(math.pow(2, 3))        # 2^3 = 8.0
print(math.pi)               # 3.141592...

🎲 2. random Module

Used to generate random numbers, choose random items, etc.

import random

print(random.randint(1, 10))         # Random integer between 1 and 10
print(random.choice(['apple', 'banana', 'cherry']))  # Random choice
print(random.random())               # Random float between 0 and 1
random.shuffle([1, 2, 3, 4, 5])      # Shuffles list in-place

💻 3. os Module

Provides functions to interact with the Operating System.

import os

print(os.getcwd())           # Current working directory
print(os.name)               # OS name (e.g., 'nt' for Windows, 'posix' for Unix)
os.mkdir("new_folder")       # Create new folder
os.remove("file.txt")        # Delete a file

⏰ 4. datetime Module

Used for working with dates and times.

import datetime

now = datetime.datetime.now()
print("Current Date and Time:", now)

today = datetime.date.today()
print("Today's Date:", today)

delta = datetime.timedelta(days=5)
future = today + delta
print("5 days from today:", future)

💡 Tip: Import Specific Functions

from math import sqrt, pi

print(sqrt(9))   # 3.0
print(pi)        # 3.141592...

✅ User-Defined Modules in Python

🔹 What is a User-Defined Module?

A user-defined module is simply a Python file (.py) that contains functions, variables, or classes — created by you — and can be imported into other Python files.

📦 Imagine making your own toolbox that you can reuse in other scripts.

✅ Step-by-Step Example:

🔸 Step 1: Create the Module File

📝 mytools.py

# This is your user-defined module

def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

pi = 3.14159

🔸 Step 2: Use it in another file

📝 main.py

import mytools

print(mytools.greet("Aman"))      # Output: Hello, Aman!
print(mytools.add(10, 5))         # Output: 15
print(mytools.pi)                 # Output: 3.14159

✅ You can also import specific functions:

from mytools import greet, pi

print(greet("Raj"))
print(pi)

🔹 How It Works:

  • Python searches for mytools.py in the current folder or sys.path.
  • It loads the module once and caches it.
  • A .pyc (compiled) version may also be created for faster loading.

🧠 Tips:

  • File name becomes module name: mymodule.py → import mymodule
  • Avoid spaces or special characters in filenames.
  • Good for organizing large programs into smaller pieces.
