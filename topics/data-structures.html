🧠 Chapter 5: Data Structures in Python – List, Tuple, Set, Dictionary

Imagine your brain having separate compartments for different types of memory — like playlists, favorite emojis, fridge items, and contacts.
That’s what data structures in Python help you do — store and organize data efficiently! 📦📚

📦 What are Data Structures?

They’re ways of storing and organizing data so you can access, modify, and manage it easily.

Four Main Built-in Data Structures:

List✅ Yes✅ Yes✅ Yes[1, 2, 3]

🧃 1. List – A Basket of Items 🧺

A List is a mutable, ordered collection of items.

fruits = ["apple", "banana", "mango"]
print(fruits[1])  # Output: banana

🎯 Useful List Methods:

fruits.append("orange")  # Add item
fruits.remove("apple")   # Remove item
fruits.sort()            # Sort list
len(fruits)              # Count items
😄 Funny Analogy:
A list is like your shopping bag 🛍️—you can add more stuff, remove old stuff, and even sort it!

🪑 2. Tuple – The Unchangeable Chair Set

A Tuple is a fixed (immutable), ordered collection.

colors = ("red", "green", "blue")
print(colors[0])  # Output: red

You can’t change a tuple once it's created — no .append() or .remove().

✅ Use When:

  • You want a constant set of values
  • Data should not be changed

🤓 Real Example:

days = ("Monday", "Tuesday", "Wednesday")
🧠 Joke:
A tuple is like your school time-table 📅—you can't change it once it's printed!

🔁 3. Set – A Unique Collection 🌟

A Set is an unordered collection of unique items.

numbers = {1, 2, 2, 3, 4}
print(numbers)  # Output: {1, 2, 3, 4}

📌 Key Points:

  • No duplicates allowed
  • Order is not guaranteed
  • Very fast for searching

🎮 Example:

set1 = {1, 2, 3}
set2 = {3, 4, 5}
print(set1.union(set2))        # {1, 2, 3, 4, 5}
print(set1.intersection(set2)) # {3}

🤖 4. Dictionary – Your Phone Contacts 📱

A Dictionary is an unordered collection of key-value pairs.

student = {
    "name": "Aman",
    "age": 21,
    "marks": 88
}
print(student["name"])  # Output: Aman

🎯 Useful Dictionary Methods:

student["course"] = "Python"     # Add new key
student.pop("age")               # Remove key
print(student.keys())            # Get all keys
print(student.values())          # Get all values
😆 Funny Analogy:
Dictionary is like your contact list — Name = Key, Phone Number = Value

💡 Real Example: Store Student Info

students = {
    "Aman": {"age": 21, "course": "Python"},
    "Riya": {"age": 20, "course": "Java"}
}

🧠 Bonus Tip:

Want to convert between structures?
✅ Yes, you can!

list1 = [1, 2, 2, 3]
set1 = set(list1)          # Remove duplicates
tuple1 = tuple(list1)      # Make immutable
