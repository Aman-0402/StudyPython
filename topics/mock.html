        self.brand = brand  # instance variable

    • c1 = Car("BMW")
    • c2 = Car("Tesla")
  3. The critical line: c2.wheels = 6
    • Creates a new instance variable wheels just for c2
    • Does not modify the class variable

Object wheels Value Explanation
c1 4 Uses class variable since no instance variable exists
c2 6 Uses its own instance variable that shadows the class variable

üí° Key Concepts:
