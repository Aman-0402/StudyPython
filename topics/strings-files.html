üìò Part 1: String Handling

Everything in the digital world involves text ‚Äî from chatting on WhatsApp to writing passwords.

üî§ What is a String?

A string is a sequence of characters:

name = "Aman"
quote = "Python is fun!"

Strings are immutable.

üß™ String Indexing & Slicing

text = "Python"
print(text[0])      # P   (first character)
print(text[-1])     # n  (last character)
print(text[1:4])    # yth (2nd to 4th character)
print(text[::-1])   # nohtyP (reversed)

üí° Indexing Recap

Index P y t h o n
Positive 0 1 2 3 4 5
Negative -6 -5 -4 -3 -2 -1

üîß String Slicing Syntax

string[start:end:step]
Parameter Description
start index to begin from (inclusive)
end index to stop before (exclusive)
step how many steps to skip (default 1)

üîß Common String Methods ‚Äì Explained with Examples

1. .lower() ‚Äì Convert to lowercase

‚úÖ Converts all characters in a string to lowercase.

text = "PYTHON"
    print(text.lower())  # Output: python

2. .upper() ‚Äì Convert to uppercase

‚úÖ Converts all characters to uppercase.

greeting = "hello"
    print(greeting.upper())  # Output: HELLO

3. .strip() ‚Äì Remove leading & trailing spaces

‚úÖ Removes unwanted spaces from both ends.

name = "   Aman   "
    print(name.strip())  # Output: 'Aman'

4. .replace() ‚Äì Replace characters/substrings

‚úÖ Replaces parts of the string with something else.

fruit = "apple"
    print(fruit.replace("p", "b"))  # Output: abble

5. .find() ‚Äì Find the index of a substring

‚úÖ Returns the first index of a substring, or -1 if not found.

msg = "hello"
    print(msg.find("e"))  # Output: 1

6. .split() ‚Äì Split string into a list

‚úÖ Splits the string by the given delimiter.

items = "a,b,c"
    print(items.split(","))  # Output: ['a', 'b', 'c']

7. .join() ‚Äì Join list into a string

‚úÖ Joins elements of a list into a single string using a separator.

letters = ['a', 'b', 'c']
    print("-".join(letters))  # Output: a-b-c

8. .isdigit() ‚Äì Check if string is numeric

‚úÖ Returns True if all characters in the string are digits.

num = "12345"
    print(num.isdigit())  # Output: True
    
    word = "abc123"
    print(word.isdigit())  # Output: False

üìò Part 2: File I/O (Input/Output)

File handling lets your Python program read from or write to external files like .txt, .csv, .json, etc.

üìù 1. Opening a File

Use the open() function to access a file.

file = open("notes.txt", "r")  # Open in read mode
  content = file.read()          # Read content
  print(content)
  file.close()                   # Close file

"notes.txt" is the file name.

"r" means "read mode".

üîì 2. File Modes

Mode Description
r Read (default)
w Write (overwrites existing file)
a Append (adds to file)
r+ Read and write

üñäÔ∏è 3. Writing to a File

file = open("notes.txt", "w")  # Open in write mode
  file.write("Learning Python is fun!\n")
  file.close()

This creates a new file or overwrites it if it already exists.

‚ûï Appending to a File

file = open("notes.txt", "a")  # Open in append mode
  file.write("Let's master file handling.\n")
  file.close()

Appends content to the end of the file without deleting existing data.

üìñ 4. Reading Line-by-Line

file = open("notes.txt", "r")
  for line in file:
      print(line.strip())
  file.close()

.strip() removes \n from each line when printed.

‚úÖ 5. Using with (Best Practice)

with open("notes.txt", "r") as f:
      content = f.read()
      print(content)

Automatically closes the file.

Preferred for safety and cleaner syntax.

üìÇ 6. Student Data Example

with open("student.txt", "w") as f:
      f.write("Name: Aman\n")
      f.write("Course: Python\n")
      f.write("Marks: 88\n")

To read it:

with open("student.txt", "r") as f:
      print(f.read())

üîÑ 7. User Input to File

data = input("Enter something to save: ")
  with open("mydata.txt", "w") as file:
      file.write(data)

This program lets the user type text, and saves it to a file.
