🧠 Chapter 6: String Handling & File I/O in Python

📘 Part 1: String Handling

Everything in the digital world involves text — from chatting on WhatsApp to writing passwords.

🔤 What is a String?

A string is a sequence of characters:

name = "Aman"
quote = "Python is fun!"

Strings are immutable.

🧪 String Indexing & Slicing

text = "Python"
print(text[0])      # P
print(text[-1])     # n
print(text[1:4])    # yth
print(text[::-1])   # nohtyP

🔧 Common String Methods

MethodDescriptionExample
.lower()Lowercase"PY".lower() → 'py'
.upper()Uppercase"hi".upper() → 'HI'
.strip()Remove spaces" Hello ".strip() → 'Hello'
.replace()Replace text"apple".replace("p", "b")
.find()Find index"hello".find("e") → 1
.split()Split string"a,b,c".split(",")
.join()Join list",".join(['a','b']) → 'a,b'
.isdigit()Check digits"123".isdigit() → True

✂️ Example – Remove Spaces and Capitalize

user_input = "  python is awesome "
cleaned = user_input.strip().title()
print(cleaned)  # Python Is Awesome

🔐 Real-Life Use Case

password = input("Enter password: ")
if password.strip() == "OpenSesame":
    print("Access Granted ✅")
else:
    print("Access Denied 🚫")

📘 Part 2: File I/O (Input/Output)

📝 Opening a File

file = open("notes.txt", "r")
content = file.read()
print(content)
file.close()

🔓 File Modes

ModeMeaning
rRead (default)
wWrite (overwrites)
aAppend
r+Read & Write

🖊️ Writing & Appending

# Write
file = open("notes.txt", "w")
file.write("Learning Python is fun!\n")
file.close()

# Append
file = open("notes.txt", "a")
file.write("Let's master file handling.\n")
file.close()

📖 Reading Line-by-Line

file = open("notes.txt", "r")
for line in file:
    print(line.strip())
file.close()

✅ Using with (Best Practice)

with open("notes.txt", "r") as f:
    content = f.read()
    print(content)

📂 Student Data File

# Write
with open("student.txt", "w") as f:
    f.write("Name: Aman\n")
    f.write("Course: Python\n")
    f.write("Marks: 88\n")

# Read
with open("student.txt", "r") as f:
    print(f.read())

🔄 User Input to File

data = input("Enter something to save: ")
with open("mydata.txt", "w") as file:
    file.write(data)
