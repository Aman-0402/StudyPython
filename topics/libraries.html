🔢 NumPy Basics

🛠️ Importing and Creating Arrays

import numpy as np
arr = np.array([1, 2, 3, 4])
print(arr)  # [1 2 3 4]

🧮 Array Operations

print(arr + 5)        # [6 7 8 9]
print(arr * 2)        # [2 4 6 8]
print(np.mean(arr))   # Average: 2.5
print(np.sqrt(arr))   # Square roots

📏 2D Arrays (Matrix)

matrix = np.array([[1, 2], [3, 4]])
print(matrix)

🐼 pandas Basics

📥 Importing pandas and Creating DataFrames

import pandas as pd
data = {
  "Name": ["Aman", "Riya", "John"],
  "Marks": [90, 85, 88]
}
df = pd.DataFrame(data)
print(df)

🔍 Accessing Data

print(df["Name"])     # Column
print(df.loc[0])      # First row
print(df.iloc[1])     # Second row (by index)

📈 Descriptive Stats

print(df.describe())      # Summary stats
print(df.mean(numeric_only=True))   # Average of marks

📤 Reading from CSV

df = pd.read_csv("students.csv")
print(df.head())
🤪 Funny Analogy:
NumPy is like your math nerd friend 🤓 who calculates fast.
pandas is your organized Excel-using friend who loves neat rows and columns 🧾

🔎 Real-Life Example: Analyzing Class Marks

import pandas as pd
data = {
  "Student": ["Aman", "Riya", "Rahul"],
  "Math": [95, 85, 75],
  "Science": [88, 92, 80]
}
df = pd.DataFrame(data)
print("Average Marks:")
print(df.mean(numeric_only=True))
